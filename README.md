# 一，标题

语法：#（一级标题）##（二级标题）###（三级标题）……

快捷键:

1.Ctrl+数字1~6可以快速将选中的文本调成对应级别的标题

2.Ctrl+0可以将选中的文本调成普通文本

3.Ctrl+加号/减号对标题级别进行加减

# 二，段落

### 1，换行

按回车键默认两步换行

按Shift+Enter键
完成一步换行
可在源代码中查看区别（</>)

### 2,分割线

1.按三个减号（-）+Enter键，如下：

---

2.按三或四个星号（*)+Enter键，如下：

****

# 三，文字显示

### 1，字体

语法：

1.粗体：用一对双星号包裹（**   **）

2.删除线：用一对双飘号包裹（~~  ~~）

3.下划线：用一对u包裹（<u>    </u>）

4.斜体：用一对单星号号包裹（*    *）

5.高亮：用一对双等号包裹（==  ==）



**这是粗体**
~~这是删除线~~
<u>这是下划线</u>

*这是斜体*
==这是高亮==

快捷键：

1.加粗：Ctrl+B
2.删除线：Shift+Alt+5
3.下划线：Ctrl+U
4.斜体：Ctrl+L

### 2，上下标

1.上标（^2^)
x^2^

2.下标（~2~）
H~2~o
